Deep Dive — Auto Mode

Der intelligente Mittelweg.

Bisher gab es zwei Extreme: Jede Aktion bestätigen — oder alles blind erlauben. Auto Mode nutzt einen Echtzeit-Klassifikator, der jede geplante Aktion auf ihr Risiko prüft und nur bei echten Gefahren stoppt.

Claude fragt bei jeder Dateiänderung und jedem Terminal-Befehl um Erlaubnis. Sicher, aber der Workflow pausiert alle paar Sekunden. Keine Möglichkeit, eine große Aufgabe zu delegieren und kurz wegzugehen.

Auto Mode
Ein integrierter Klassifikator prüft jede Aktion in Echtzeit. Ungefährliche Aktionen (Code schreiben, Dateien erstellen, Tests ausführen) laufen automatisch. Riskante Befehle erfordern weiterhin manuelle Freigabe.
Neu
Use Cases

Wo Auto Mode den Unterschied macht.

01
Großflächiges Refactoring
40 Dateien müssen ein neues Design-Pattern bekommen. Im normalen Modus: 40-mal manuell bestätigen. Im Auto Mode stuft der Klassifikator reine Datei-Updates innerhalb des Projektordners als sicher ein — Claude arbeitet alles in einem Rutsch durch.
02
Autarke Debugging-Loops
Claude analysiert Code, passt ihn an, führt Tests aus, liest Fehlermeldungen, passt erneut an. Der gesamte iterative Prozess läuft ohne Unterbrechung, weil lokale Test-Skripte als sicher eingestuft werden — bis alle Tests grün sind.
03
Projekt-Setup & Boilerplate
Ordnerstruktur, package.json, Datenbankkonfiguration, API-Routen — das massenhafte Erstellen und Befüllen leerer Dateien gilt als ungefährlich. Claude baut das komplette Projektgerüst in Sekunden, ohne jede einzelne Datei abzunicken.
04
Datenaufbereitung & Skripte
100 CSV-Dateien einlesen, bereinigen, zusammenfassen. Claude schreibt das Skript, testet es an einer Beispieldatei, prüft den Output und bessert Fehler aus — alles im Hintergrund ohne manuelle Zwischenschritte.

08 — FAQ

Häufig gestellte Fragen.

Was ist Adaptive Thinking in Opus 4.6?
Anstatt manuell ein Token-Budget für das Nachdenken festzulegen, entscheidet Claude selbst, wie intensiv es über eine Aufgabe nachdenkt. Aktiviert mit thinking: {type: "adaptive"}. Einfache Fragen werden schnell beantwortet, komplexe Probleme erhalten automatisch tiefere Analyse.
Was kann Claude Cowork konkret?
Cowork laeuft als Desktop-Agent in einer isolierten VM. Es kann Dateien lesen, bearbeiten und erstellen, Tabellenkalkulationen aus Rohdaten generieren, Downloads organisieren und Routineaufgaben automatisieren. Mit Computer Use (seit 23. März) steuert es auch Tastatur und Maus direkt.
Was ist der Pentagon-Konflikt?
Anthropic zog zwei rote Linien: keine Massenüberwachung und keine autonomen Waffen. Die Trump-Regierung stufte Anthropic daraufhin als Lieferkettenrisiko ein. Die öffentliche Reaktion war massiv pro-Anthropic — Claude stieg zur Nr. 1 in beiden App Stores auf.
Was kostet Claude aktuell?
Free: Sonnet 4.6, begrenzte Nachrichten. Pro ($20/Monat): Opus 4.6, 5x mehr Nutzung, Prioritaet. Max ($100-200/Monat): Cowork, Computer Use, Claude Code. Teams ($25/User): Workspaces, Admin-Controls. Enterprise: SSO, SCIM, HIPAA-konform.
Was hat sich bei der API geändert?
Structured Outputs, Web Search, Code Execution und Memory Tool sind jetzt GA (kein Beta-Header). Neu: Compaction API für endlose Konversationen, Data Residency Controls, automatisches Caching, Dynamic Filtering für Web-Suche. Konsole migriert zu platform.claude.com.
